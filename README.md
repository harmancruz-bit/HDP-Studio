# HDP Studio - Workspace UI

This branch contains the UI scaffold for HDP Studio built with React + TypeScript intended to be wrapped by Tauri.

What is included
- React + TypeScript scaffold (Vite)
- Top navigation bar, left sidebar, center workspace canvas, right inspector panel, bottom panel
- Dark theme and responsive layout
- Reusable component files (placeholders)

Notes
- This commit only contains UI structure, no business logic.
- To run the web UI:
  - npm install
  - npm run dev
